Mre_ini_parser

App demonstrate how to use ini file as config data source. Autoload data if app and ini file with name identifical app name is on same directory. LEFT_SOFTKEY - browse ini file, RIGHT_SOFTKEY - exit app, KEY_NUM1 - test1, KEY_NUM2 - test2. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application file - "mre_ini_parser.vxp"
